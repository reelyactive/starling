starling
========


An emulator to feed barnowl and barnacles
-----------------------------------------

Starlings imitate a variety of avian species and have a repertoire of about 15â€“20 distinct imitations. They also imitate a few sounds other than those of wild birds.


Installation
------------

    npm install starling


Hello starling
--------------

```javascript
var starling = require('starling');
var emulator = new starling();
```
